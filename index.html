Key Features

Raycasting Algorithm

Accurate wall detection using optimized raycasting to create realistic 3D maze walls.

Player Movement & Collision

Smooth WASD player controls and collision detection ensure a challenging experience.

Textured Environment

Immersive environment rendering with walls and textures, powered by SDL2.

About the Project

The Maze Game was inspired by my interest in exploring game development with a focus on performance optimization. After encountering numerous challenges with SDL2 installation and raycasting algorithms, I was determined to overcome these technical hurdles. This project is part of my portfolio for Holberton School.
